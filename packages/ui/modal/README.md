# @srcube-taro/modal

Modal component.

Refer the [docs](https://srcube-taro.vercel.app/docs/components/modal)

## Installation

```bash
pnpm add @srcube-taro/modal
# or
yarn add @srcube-taro/modal
# or
npm i @srcube-taro/modal
```

#### Tip

- The modal `z-index` is `1000` by default.
