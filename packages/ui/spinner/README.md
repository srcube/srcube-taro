# @srcube-taro/spinner

Spinner component, loading or pending status.

Refer the [docs](https://srcube-taro.vercel.app/docs/components/spinner)

## Installation

```bash
pnpm add @srcube-taro/spinner
# or
yarn add @srcube-taro/spinner
# or
npm i @srcube-taro/spinner
```
