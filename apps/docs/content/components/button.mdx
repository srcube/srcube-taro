## Advanced Usage

- `isLoading="auto"` is recommended for async onTap scenarios, as it will automatically handle the loading state.
- `startContent` and `endContent` allow you to add custom content before or after the button text (such as icons or badges).
- Button can inherit color, size, variant, isDisabled, isLoading, etc. from ButtonGroup, but individual Button props take precedence.
- Controlled loading is suitable for global loading management, while auto loading is ideal for local async actions.
